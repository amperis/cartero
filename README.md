## CARTERO 

SMTP debug tool (from, recipients, data, attachment, attachment virus, delay commands). Search MX register from domain (IP, MX number, ping, time, spam, open SMTP port). Search whois domain.

### Requeriments

* Visual Studio 2005 and +
